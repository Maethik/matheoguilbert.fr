	<h2>La mission</h2>

	<ProjectsColumns>
		<div>
			GGCG est une application front-end qui agrège les données de contribution Git de multiples sources. Le projet a vocation à être auto-hébergé par les développeurs qui le souhaitent. Une application mobile existe aussi.
		</div>
		<div>
			Ce projet a posé plusieurs défis. Entre autres, la conception de l'architecture de l'application pour concilier performances des agrégations de données coûteuses et répétitives, disponibilité du service auto-hébergé par l'utilisateur ou publiquement accessible via l'application mobile, persistance des configurations utilisateur et respect des quotas API des différentes forges.  
			Le tout en maîtrisant la confidentialité des données et leur niveau de sensibilité.
		</div>
	</ProjectsColumns>
